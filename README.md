# Financial-Analysis-Project

Dieses Projekt dient der Analyse von Zeitreihen und der Prognose von Umsatzdaten mittels Vektorautoregressiven Modellen (VAR). 

### Ziel des Projekts
- Anwendung statistischer Methoden auf Finanzdaten.
- Implementierung von VAR-Modellen zur Identifikation von Abhängigkeiten.
- Visualisierung von Prognoseergebnissen zur Unterstützung datengestützter Entscheidungen.

### Verwendete Technologien
- Python (Pandas, Statsmodels, Matplotlib)
- Jupyter Notebook
